# urenscan-pilot
UrenScan is een SaaS-applicatie waarmee medewerkers eenvoudig hun gewerkte uren kunnen registreren door een foto te maken van een urenbon of handgeschreven tijdschrijfformulier. De AI leest de gegevens automatisch uit en vult deze direct in een digitaal systeem in. Leidinggevenden kunnen de uren realtime inzien en goedkeuren of afkeuren.
